Comandos Básicos de Git

git init

Inicializa un nuevo repositorio de Git.

git init
git status

Muestra el estado del repositorio de trabajo y el área de preparación.

git status

Comandos de Branching en Git

git branch

Lista, crea o elimina ramas.

git branch
git checkout

Cambia a una rama específica.

git checkout nombre-de-la-rama

Comandos Remotos de Git

git remote

Gestiona los repositorios remotos.

git remote
git fetch

Descarga los objetos y referencias de otro repositorio.

git fetch
